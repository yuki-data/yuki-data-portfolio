About

エンジニア転職を目指している元フリーターです。
現在プログラミングスクールTECH::EXPERTでRuby on railsを勉強中です。
独学でPythonも勉強中です。

使用経験のある技術、言語、ライブラリ、ツールなど
  • 言語： HTML (HAML, ERB), CSS (SASS), JavaScript (jQuery), Ruby, Python
  • アプリケーションフレームワーク: Ruby on rails (Ruby), Flask (Python)
  • インフラ: AWS(EC2, S3), Heroku
  • データベース: MySQL
  • Ruby gem: Capistrano (自動デプロイ), carrierwave (画像投稿), devise (ログイン認証), rspec (ユニットテスト)など
  • Pythonライブラリ: numpy (行列演算), matplotlib (グラフ作成), pandas (表形式データ処理)など

Works

chatspace

chatspaceはグループチャットアプリです。画像とテキストの投稿ができ、グループに登録したメンバーのみがチャットを閲覧できます。

詳しい使い方はgithubでスクリーンショット付きで示しています。

使用した技術、言語、ツールなど

  • 言語
    • サーバーサイド: Ruby
    • フロントエンド: SASS, HTML (HAML), JavaScript (jQuery)
  • アプリケーションフレームワーク: Ruby on rails
  • インフラ: AWS(EC2, S3)
  • webサーバー: nginx
  • アプリケーションサーバー: unicorn
  • データベース: MySQL

実装内容

  • HAMLとSASSでのフロント実装
  • jQueryによるメッセージの非同期通信
  • jQueryによるインクリメンタルサーチ機能
  • devise (認証用gem)によるユーザー登録・ログイン機能
  • carrierwaveによる画像のアップロード
  • fogによるAWS S3への画像のアップロード
  • capistranoによる自動デプロイ
Github

Twitter Trend Ranking

Twitterのトレンドトピックをランキング形式で表示するwebサイトです。現在の日本のトレンドデータをtwitter APIで取得しています。

使用した技術、言語、ツールなど

  • 言語
    • サーバーサイド: Python
    • フロントエンド: HTML, CSS, JavaScript (jQuery)
  • アプリケーションフレームワーク: Flask
  • CSSフレームワーク: Bootstrap
  • インフラ: Heroku
  • アプリケーションサーバー: unicorn
  • API: twitter API

実装内容

  • tweepy (twitter API用のPythonラッパーライブラリ)でtwitterのトレンドデータを取得
  • PythonライブラリのFlaskでwebアプリケーションを作成
  • Herokuへのデプロイ。APIキーの設定。
Github ランキングサイト

Webscraping tool for google finance

google financeの株価データを取得するデスクトップツールです。以前はAPIがgoogleにより提供されていましたが、廃止されたため、スクレイピングにより取得するツールを作りました。

使用した技術、言語、ツールなど

  • 言語: Python
  • ライブラリ
    • Matplotlib: 株価チャートの作成
    • Requests: webスクレイピングでgoogleファイナンスから株価データを取得
    • BeautifulSoup: HTMLファイルの構文解析
    • Pandas: 表形式データの数値処理

ツールの使用方法はgithubのreadmeに書きました。
