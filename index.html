About

エンジニア転職を目指している元フリーターです。
現在プログラミングスクールTECH::EXPERTでRuby on railsを勉強中です。
独学でPythonも勉強中です。

使用経験のある技術、言語、ライブラリ、ツールなど
  • 言語： HTML (HAML, ERB), CSS (SASS), JavaScript (jQuery), Ruby, Python
  • アプリケーションフレームワーク: Ruby on rails (Ruby), Flask (Python)
  • インフラ: AWS(EC2, S3), Heroku
  • データベース: MySQL
  • Ruby gem: Capistrano (自動デプロイ), carrierwave (画像投稿), devise (ログイン認証), rspec (ユニットテスト)など
  • Pythonライブラリ: numpy (行列演算), matplotlib (グラフ作成), pandas (表形式データ処理)など

ツイッターのクローンアプリです。プログラミングスクール卒業後に個人で制作したWebアプリです。

詳しい使い方はgithubでスクリーンショット付きで示しています。

使用した技術、言語、ツールなど

  • 言語
    • サーバーサイド: Ruby
    • フロントエンド: SASS, HTML (ERB), JavaScript (jQuery)
  • アプリケーションフレームワーク: Ruby on rails
  • インフラ: Heroku, AWS(S3)
  • データベース: PostgreSQL

実装内容

  • ERBとSASSでフロント実装
  • CSSフレームワークとしてBootstrapを使用
  • Bootstrapでモーダルウィンドウを実装し、記事投稿・編集をモーダルウィンドウで行えるようにした
  • 非同期通信時に、jQueryにより記事やブックマークをビューに反映
  • devise (認証用gem)によるユーザー登録・ログイン機能
  • carrierwaveによる画像のアップロード
  • kaminari (ページネーション用gem)で記事数が多い際のページの切り替えを実装
  • data-confirm-modal (モーダルダイアログ用gem)により、モーダルウィンドウでの記事削除を実装
  • fogによるAWS S3への画像のアップロード
  • Herokuへのデプロイ

商品出品での画像プレビュー
商品出品

プログラミングスクールTECH::EXPERTにてチームで開発したアプリです。ユーザー登録、商品出品・購入ができるフリーマーケットアプリです。

使用した技術、言語、ツールなど

  • 言語
    • サーバーサイド: Ruby
    • フロントエンド: SASS, HTML (HAML), JavaScript (jQuery)
  • アプリケーションフレームワーク: Ruby on rails
  • インフラ: AWS(EC2, S3)
  • webサーバー: nginx
  • アプリケーションサーバー: unicorn
  • データベース: MySQL
  • チームの進捗・タスク管理: Trello, Slack

自身が担当した機能・実装内容

  • 商品情報表示（フロント・サーバーサイド両方）
    • slick.js (JavaScriptライブラリ)で商品サムネイル画像のスライドショーを実装
    • データベースから既存の商品情報を取り出して表示するサーバーサイドでの機能を実装
  • 商品情報編集(フロント・サーバーサイド両方)
    • 既存の商品情報を更新するサーバーサイドでの機能を実装
  • 商品出品(フロント・サーバーサイド両方)
    • 商品情報の入力フォームをHAML, SCSSで実装
    • 商品画像のアップロード機能をcarrierwaveで実装。
    • 複数の商品画像のプレビュー表示・削除機能をjQueryで実装
    • カテゴリー選択フォームをajaxにより動的に追加する機能をjQueryで実装
  • 商品購入(サーバーサイド)
    • Payjpによるクレジットカード決済機能の実装

Webscraping tool for google finance

google financeの株価データを取得するデスクトップツールです。以前はAPIがgoogleにより提供されていましたが、廃止されたため、スクレイピングにより取得するツールを作りました。

使用した技術、言語、ツールなど

  • 言語: Python
  • ライブラリ
    • Matplotlib: 株価チャートの作成
    • Requests: webスクレイピングでgoogleファイナンスから株価データを取得
    • BeautifulSoup: HTMLファイルの構文解析
    • Pandas: 表形式データの数値処理

工夫した点・注目してほしい点

  • 独学したPythonで作成したアプリです。
  • 株価データを単にダウンロードするだけでなく、pandasのdataframeの形式にすることで、その後のチャート作成にスムーズに取り組めるようにしました
  • 複数の銘柄のデータを連続してcsvとしてダウロードする機能も実装しました

ツールの使用方法はgithubのreadmeに書きました。
